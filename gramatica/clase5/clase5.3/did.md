# Clase: DID en Inglés

## 1. Introducción
En inglés, **DID** es un **verbo auxiliar** que se utiliza principalmente para **formar preguntas, negaciones y dar énfasis** en el **pasado simple**.  

La importancia de DID radica en que **permite usar cualquier verbo en pasado sin necesidad de cambiar su forma**, excepto para dar énfasis o negar.

- Ejemplo sin DID:  
  - I went to the park. → Factual, afirmación simple.
- Ejemplo con DID (énfasis o pregunta):  
  - Did I go to the park? → Pregunta  
  - I **did** go to the park! → Énfasis  

---

## 2. Uso de DID

### 2.1 Para formar preguntas
La estructura es:

Did + sujeto + verbo base + ?


Ejemplos:
- Did you watch the movie last night? → ¿Viste la película anoche?  
- Did she call her mother yesterday? → ¿Llamó ella a su madre ayer?  

**Nota:** El verbo principal siempre está en **forma base**, aunque la acción esté en pasado.  

---

### 2.2 Para formar negaciones
La estructura es:

Sujeto + did not (didn't) + verbo base


Ejemplos:
- I didn’t see him yesterday. → No lo vi ayer.  
- They didn’t finish their homework. → No terminaron su tarea.  

---

### 2.3 Para dar énfasis
Podemos usar DID para **resaltar o confirmar una acción pasada**.  

Estructura:

Sujeto + did + verbo base


Ejemplos:
- I **did** call you last night! → ¡Sí te llamé anoche!  
- She **did** enjoy the concert. → Ella sí disfrutó el concierto.  
- We **did** complete the project on time. → Sí completamos el proyecto a tiempo.  

**Tip:** El énfasis con DID se usa para reforzar la acción, generalmente en respuesta a dudas o negaciones.  

---

## 3. Resumen rápido

| Función       | Estructura                   | Ejemplo                            |
|---------------|------------------------------|------------------------------------|
| Pregunta      | Did + sujeto + verbo base ?  | Did you study for the test?        |
| Negación      | Sujeto + did not + verbo base| I didn’t eat breakfast yesterday.  |
| Énfasis       | Sujeto + did + verbo base    | I did finish my homework!          |
