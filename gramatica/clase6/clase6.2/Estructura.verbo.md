# Clase: Estructuras Modales y de Intención en Inglés

## 1. Introducción
En inglés, existen **verbos modales y frases de intención** que se usan para expresar **habilidad, posibilidad, obligación, intención, permiso o deseo**.  
Algunos ejemplos comunes incluyen: **I can, I want to, I need to, I should, I would like to, I have to, I might, I must, I am going to, I plan to**.

---

## 2. Lista de Modales y Frases de Intención

| Estructura        | Uso / Significado                               | Ejemplo |
|------------------|-----------------------------------------------|---------|
| I can            | Habilidad o posibilidad                         | I can swim. → Puedo nadar. |
| I want to        | Deseo / intención                               | I want to learn English. → Quiero aprender inglés. |
| I need to        | Necesidad / obligación                          | I need to finish my homework. → Necesito terminar mi tarea. |
| I should         | Consejo / recomendación                         | I should study more. → Debería estudiar más. |
| I would like to   | Deseo formal / preferencia                       | I would like to travel to Japan. → Me gustaría viajar a Japón. |
| I have to        | Obligación / necesidad                           | I have to go to work. → Tengo que ir al trabajo. |
| I might          | Posibilidad / probabilidad                       | I might visit my friend tomorrow. → Podría visitar a mi amigo mañana. |
| I must           | Obligación fuerte / necesidad                     | I must finish this project today. → Debo terminar este proyecto hoy. |
| I am going to    | Intención futura / plan                           | I am going to call my mom. → Voy a llamar a mi mamá. |
| I plan to        | Planificación / intención                         | I plan to study tonight. → Planeo estudiar esta noche. |

---

## 3. Tipos de uso
1. **Habilidad y posibilidad:** can, might  
2. **Deseo e intención:** want to, would like to, am going to, plan to  
3. **Obligación y necesidad:** need to, must, have to  
4. **Consejo o recomendación:** should
