### dígrafos vocálicos
- **ee** → /se pronuncia como I/  (see, street, feet)
- **ea** → /se pronuncia como I/  (eat, team, sea)
- **ai** → /se pronuncia como EI/  (rain, train, wait)
- **aw** → /se pronuncia como O/  (saw, draw, law)
- **oo** → /se pronuncia como U/ o /ʊ/ (moon vs book)
- **ou** → /se pronuncia como AU/  (out, sound, cloud)

### trígrafo
- **igh** → /se pronuncia como AI/ (night, light, right)
